# Attendance Management System (Face Detection)

## Getting Started
1. **Clone the repo:** `git clone <repo_url>`
2. **Backend Team:** Go to `backend/` folder. Read the comments in `app.py`.
3. **Database Team:** Go to `database_scripts/`. Read `init_db.sql`.
